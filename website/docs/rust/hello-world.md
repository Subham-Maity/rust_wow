# ▶️ Run Program

> create a file `main.rs`

```rust
fn main() {
    println!("Hello, world!");
}
```
Open your terminal 
- mac/linux
```shell
PS \Rust_WOW> rustc main.rs
PS \Rust_WOW> .\main
Hello, world!
```
- windows
```shell
PS \Rust_WOW> rustc main.rs
PS \Rust_WOW> .\main.exe
Hello, world!
```
> - <Highlight> rustc is the compiler</Highlight>
